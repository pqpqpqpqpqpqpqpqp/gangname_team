디자인 개선
Soft Reset
변경사항을 스테이징 영역에 유지
git reset --soft HEAD~1
Mixed Reset
변경사항을 작업 디렉토리에 유지
git reset --mixed HEAD~1
Hard Reset
모든 변경사항 삭제
git reset --hard HEAD~1
